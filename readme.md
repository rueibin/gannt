
## dhtmlxGantt 甘特圖
功能可以建立、編輯、刪除與調整任務順序。

1.下載程式碼
- git clone https://github.com/rueibin/gannt.git

2.進入專案目錄，之後composer安裝
- composer install

3.配置.evn
- 將.env.example複製成.env

4.配置資料庫
- DB_HOST=localhost
- DB_DATABASE=homestead
- DB_USERNAME=homestead
- DB_PASSWORD=secret

5.遷移資料
- php artisan migrate --seed
